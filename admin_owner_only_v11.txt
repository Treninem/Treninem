ПАНЕЛЬ ВЛАДЕЛЬЦА «ЗВЕРОБОЙ» — КРАТКО

Вход:
1) только в ЛС бота
2) команда /adm
3) либо /adm <пароль>

Главные разделы:
- 💎 Паки — список паков, включение/выключение обычной продажи
- 💫 Донат — глобально включить/выключить донат, включить/выключить продажу каждого пака за звёзды, задать цену в звёздах
- 🎟 Промокоды — создание промокодов
- 🚨 Подозрения — сигналы по мультиаккаунтам и подозрительным переводам
- 🆕 Новый предмет — создание предмета через ЛС
- 🧪 Новый крафт — создание рецепта через ЛС
- ✏️ Изменить предмет — редактирование параметров предмета на русском
- 🎁 Выдать пак — по @username, ID или ответом на сообщение
- 🎒 Выдать предмет — по @username, ID или ответом на сообщение
- 🗑 Забрать предмет — по @username, ID или ответом на сообщение
- 🏦 Долг банка — установить, уменьшить или аннулировать банковский долг игрока
- ⭐ Уровень — изменить уровень игрока
- ⛔ Блок / ✅ Разблок — блокировка игрока
- 👮 Админ + / 👤 Админ - — выдать или снять права администратора

Промокоды:
Формат создания:
КОД КОЛИЧЕСТВО_АКТИВАЦИЙ ЗОЛОТО ПРЕМИУМ

Пример:
VESNA2026 50 100 2

Это означает:
- код: VESNA2026
- активаций: 50
- награда: 100 монет и 2 премиум-жетона

Как вводит игрок:
- игрок вводит только сам код
- без слов uses/gold/premium

Пример для игрока:
VESNA2026

Долг банка:
Формат:
@username сумма

Примеры:
@Treninem 0        -> полностью аннулировать долг
@Treninem -500     -> уменьшить долг на 500
@Treninem 1200     -> установить долг ровно 1200

Цена пака в звёздах:
Формат:
код_пака цена_в_звёздах

Пример:
starter_pack 35

Действия по игрокам работают так:
- по @username
- по Telegram ID
- ответом на сообщение
