ЗВЕРОБОЙ 3.0 APEX — сборка для Bothost

Что нового в этой версии:
- максимум уровня повышен до 600
- добавлены таланты и очки талантов каждые 10 уровней
- добавлены фракции с пассивными бонусами и ежедневным жалованием
- добавлены ежедневные контракты с отдельными наградами
- добавлен престиж на максимальном уровне
- усилена динамическая экономика магазина и чёрного рынка
- улучшен боевой и PvE баланс под длинный эндгейм
- owner-only админ панель остаётся скрытой для всех кроме владельца

Короткие команды:
/m /p /inv /x /d /s /bm /mk /c /b /task /boss /pet /camp /clan /mail /top /ref /info /tal /fac /ct

Запуск на Bothost:
1. Загрузи архив
2. Убедись, что установлен Python 3.10+
3. Установи зависимости из requirements.txt
4. Запусти bot.py

Важно:
- токен уже записан в .env
- так как токен был отправлен в чат, после проверки лучше перевыпустить его у BotFather и заменить в .env

Новые разделы:
- Таланты: меню -> Таланты
- Фракция: меню -> Фракция
- Контракты: меню -> Контракты
- Престиж: кнопка появляется в профиле на 600 уровне

Админ-панель:
- видна только OWNER_ID
- даже если кто-то узнает пароль, кнопки в меню для него не появятся


Структура архива очищена для Bothost: файлы лежат в корне архива, без длинных путей, без __pycache__ и без старой базы v2.
