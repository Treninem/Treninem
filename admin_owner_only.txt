Как теперь выдавать предмет:
1. ЛС бота → /adm
2. Нажми "🎒 Выдать предмет"
3. Либо выбери предмет из списка, либо напиши название вручную
4. После выбора укажи игрока ответом на сообщение, через @username или ID
5. Затем укажи количество

Как теперь забирать предмет:
1. ЛС бота → /adm
2. Нажми "🗑 Забрать предмет"
3. Либо выбери предмет из списка, либо напиши название вручную
4. После выбора укажи игрока ответом на сообщение, через @username или ID
5. Затем укажи количество

Как теперь выдавать пак:
1. ЛС бота → /adm
2. Нажми "🎁 Выдать пак"
3. Либо выбери пак из списка, либо напиши название/код вручную
4. Укажи игрока ответом на сообщение, через @username или ID

Как добавить новый предмет:
1. ЛС бота → /adm
2. Нажми "🆕 Новый предмет"
3. Отправь: item_id Название предмета
4. Бот определит категорию и редкость по ID
5. Дальше выбери шаблон действия кнопкой

Как добавить новый крафт:
1. ЛС бота → /adm
2. Нажми "🧪 Новый крафт"
3. Отправь строку:
   recipe_id result_item result_amount ingredient_id:qty,ingredient_id:qty
4. Выбери станцию кнопкой
