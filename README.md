# Bot_PUBG

Готовый Telegram-бот для клана PUBG на `python-telegram-bot 20.x` и `Python 3.10`.

## Что уже реализовано

- регистрация игроков по PUBG ID через PUBG API;
- роли: владелец, администратор чата, обычный пользователь, наставник;
- профиль, прогресс, достижения, баллы;
- наставничество, список наставников, подопечные, задания, тренировки;
- новости, события, обновления игры;
- премиум-раздел;
- друзья, рейтинг друзей;
- список групп с ботом;
- обратная связь и тикеты;
- ежедневные тесты и напоминания;
- SQLite база данных;
- логирование;
- unit/integration tests.

## Структура

Проект уже разложен по папкам и файлам согласно заданной структуре.

## Как запускать локально

### 1. Установите Python 3.10
Проверьте:

```bash
python3 --version
```

### 2. Перейдите в папку проекта

```bash
cd Bot_PUBG
```

### 3. Создайте виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
```

Windows:

```bash
venv\Scripts\activate
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Запустите бота

```bash
python main.py
```

## Как запускать на Bothost

### Вариант 1. Через Git / архив
1. Загрузите весь архив проекта на Bothost.
2. Убедитесь, что стартовая команда указана как:

```bash
python main.py
```

3. Версия Python должна быть **3.10**.
4. Установите зависимости из `requirements.txt`.

### Вариант 2. Через файловый менеджер Bothost
1. Создайте папку `Bot_PUBG`.
2. Загрузите в нее все файлы из архива.
3. В настройках приложения укажите команду запуска:

```bash
python main.py
```

4. Включите автоперезапуск процесса.

## Что и где находится

### Основной запуск
- `main.py` — главный файл запуска.

### Настройки и ключи
- `config/credentials.py` — токен бота, PUBG API key, ID владельца, база данных.
- `config/settings.py` — время тестов, таймзона, внутренние настройки.
- `config/constants.py` — ранги, баллы, лимиты, логика ролей.

### База данных
- `database/models.py` — модели SQLAlchemy.
- `database/queries.py` — работа с БД.
- `database/schema.sql` — SQL-схема для просмотра.

### Обработчики Telegram
- `handlers/*.py` — меню и команды.

### Сервисы
- `services/pubg_api.py` — запросы к PUBG API.
- `services/notifications.py` — напоминания и уведомления.
- `services/monitoring.py` — логирование.
- `services/yandex_cloud.py` — заготовка под Yandex Cloud / Object Storage.

### Фичи
- `features/daily_tests/*` — ежедневные тесты.
- `features/points_system/*` — баллы и ранги.
- `features/mentorship/*` — наставничество.

## Что еще желательно сделать перед боевым запуском

### 1. Сменить токен бота
Так как токен уже используется в коде, для безопасности после первого запуска лучше:
- открыть `@BotFather`;
- выбрать вашего бота;
- выполнить `/revoke` или `/token`;
- вставить новый токен в `config/credentials.py`.

### 2. Сменить PUBG API ключ при необходимости
Если ключ перестанет работать:
- зайдите в PUBG Developer Portal;
- создайте новый ключ;
- замените его в `config/credentials.py`.

### 3. Перевести секреты в переменные окружения
Сейчас проект запускается "из коробки". Для production лучше хранить секреты в ENV.

Пример:

```bash
export TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
export PUBG_API_KEY="ВАШ_PUBG_API_KEY"
export OWNER_ID="2097006037"
```

## Как пользоваться ботом

### Первый запуск
1. Напишите боту `/start`.
2. Откройте раздел `👤 Профиль`.
3. Нажмите `🆕 Зарегистрироваться / Редактировать`.
4. Введите PUBG ID.
5. Введите возраст.
6. Введите часы игры в неделю.

### Наставничество
1. Нажмите `👨‍🏫 Наставничество`.
2. Выберите `👨‍🏫 Стать наставником`.
3. Если ваш ранг `Корона+`, бот активирует статус после анкеты.

### Команды наставника
- `/create_training 2026-04-20_19:00 medium Тренировка по позиционке`
- `/create_task 123456789 Стрельба на дальнюю | Сделать 20 попаданий в тренировке`

### Команды владельца
- `/p_up 123456789`
- `/p_dn 123456789`

### Команда администратора чата
- `/clear_bot_messages`

## Важные ограничения Telegram API

Telegram Bot API **не может сам создать обычную группу** без участия человека.
Поэтому в разделе `🗣️ Чат с друзьями` бот объясняет, как создать группу вручную и добавить туда бота.
