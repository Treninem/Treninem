# Bot_PUBG

Полностью готовый проект Telegram-бота на `python-telegram-bot 20.x` с SQLite, PUBG API, наставничеством, системой баллов, ежедневными тестами, друзьями, новостями и обратной связью.

---
## 1. Что уже встроено

В проект уже заложено:
- токен бота
- PUBG API key
- ID владельца
- username владельца
- SQLite база данных
- ежедневные тесты через `JobQueue`
- меню и все основные обработчики
- скрытые команды владельца `/p_up` и `/p_dn`
- команда админа чата `/clear_bot_messages`

## 2. Важное замечание по API и ограничениям

### PUBG API
Официальный PUBG API отлично подходит для:
- поиска игрока
- получения player id
- получения lifetime/ranked-статистики

Но есть нюансы:
- официальный PUBG API **не предоставляет отдельный news API**
- ранги в разных PUBG-продуктах отличаются по названиям
- термин «Корона» характерен не для всех версий PUBG API

Поэтому в проекте сделано так:
- **статистика игрока** идёт через PUBG API
- **новости** подтягиваются через публичный news endpoint Steam для PUBG
- **проверка наставника** сделана гибко по ключевым high-tier рангам

### Telegram Bot API
Telegram Bot API **не умеет сам создавать новый групповой чат с друзьями по кнопке**.
Поэтому раздел «Чат с друзьями» реализован корректно и честно:
- бот объясняет, как создать группу вручную
- после добавления бота в группу этот чат появится в разделе «Группы с ботом»

## 4. Быстрый запуск локально

### Шаг 1. Установи Python
Нужен Python **3.10**.

### Шаг 2. Открой папку проекта
```bash
cd Bot_PUBG
```

### Шаг 3. Создай виртуальное окружение
```bash
python -m venv venv
```

### Шаг 4. Активируй окружение

Windows:
```bash
venv\Scripts\activate
```

Linux/macOS:
```bash
source venv/bin/activate
```

### Шаг 5. Установи зависимости
```bash
pip install -r requirements.txt
```

### Шаг 6. Запусти бота
```bash
python main.py
```

---
## 5. Что куда вставлять

Все основные данные уже стоят в:
`config/credentials.py`

Там находятся:
- `BOT_TOKEN`
- `PUBG_API_KEY`
- `OWNER_ID`
- `OWNER_USERNAME`
- `OWNER_CHAT_ID`

Если захочешь заменить их позже:
1. Открой файл `config/credentials.py`
2. Замени значения
3. Сохрани
4. Перезапусти бота

---
## 6. Что лучше сделать сразу после первого запуска

Из соображений безопасности рекомендуется:
1. Перевыпустить токен бота через **@BotFather**
2. Если нужно — перевыпустить PUBG API key в кабинете разработчика
3. После этого обновить `config/credentials.py`

---
## 7. Как получить новый токен бота

1. Открой Telegram
2. Найди **@BotFather**
3. Отправь команду `/mybots`
4. Выбери нужного бота
5. Нажми **API Token**
6. Либо сгенерируй новый токен
7. Скопируй токен
8. Вставь его в `config/credentials.py`

---
## 8. Как получить PUBG API key

1. Перейди в кабинет разработчика PUBG
2. Создай приложение
3. Сгенерируй API key
4. Вставь ключ в `config/credentials.py`

### Выдать премиум
```bash

```

- `123456789` — Telegram ID пользователя
- `30` — количество дней

### Убрать премиум
```bash

```

## 13. Команда админа чата

В группе, где бот добавлен:
```bash
/clear_bot_messages
```

Она удаляет сохранённые сообщения бота из текущего чата.

---
## 14. Как устроены ежедневные тесты

После регистрации пользователю автоматически ставятся 2 ежедневные задачи:
- напоминание за 15 минут
- отправка теста в 10:00

Вопросы:
- 5 обычных
- 1 бонусный

Начисление:
- +10 за прохождение ежедневного теста
- +15 за правильный бонусный ответ

## 17. Если что-то не запускается

Проверь:
1. Python 3.10
2. установлен ли `python-telegram-bot[job-queue]`
3. не отозван ли токен бота
4. доступен ли PUBG API key
5. есть ли интернет у хоста
6. что `logs/bot.log` не показывает traceback

## 18. Рекомендации на продакшен

Для реального продакшена лучше:
- вынести токены в переменные окружения
- перевыпустить токен бота
- использовать PostgreSQL вместо SQLite при большой нагрузке
- добавить резервное копирование БД
- ограничить выдачу бонусов за приглашение друга от повторных злоупотреблений
- добавить отдельную админ-панель при масштабировании


## Улучшения очистки чата

- В личных сообщениях бот удаляет команды, ответы меню и шаги ввода пользователя.
- Бот старается держать одно актуальное меню и один актуальный экран вместо длинной ленты служебных сообщений.
- Дайджесты новостей, наставников и групп теперь приходят компактнее.
- Итоги тестов и напоминания автоматически удаляются позже, чтобы не захламлять диалог.


## Новое в этой сборке

- Привязка Telegram ID ↔ PUBG player ID.
- Бот сохраняет расширенный профиль PUBG: ник, уровень, ранг, матчи, победы, убийства, урон, топ-10, винрейт.
- Добавлены команды `/binding` и `/sync_profile`.
- Один PUBG аккаунт нельзя привязать к двум Telegram аккаунтам.
