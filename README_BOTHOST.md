# Архив для Bothost

Этот архив подготовлен в максимально простом формате для загрузки как Python-проект.

## Что делать после загрузки

1. Загрузи содержимое архива в проект на Bothost.
2. Открой файл `.env` и заполни:
   - `TELEGRAM_BOT_TOKEN`
   - `OPENAI_API_KEY`
   - `ADMIN_USER_IDS`
   - `ROOT_OWNER_USER_ID`
3. Установи зависимости из `requirements.txt`.
4. Укажи команду запуска:

```bash
python -u bot.py
```

## В архив уже добавлено

- `start.sh`
- `Procfile`
- `runtime.txt`
- `.env.example`
- `bothost_start.txt`

## Если нужен самый простой вариант

Используй:

```bash
python -u bot.py
```

## Важно

- Не запускай этот же токен параллельно где-то ещё.
- После первого старта база создастся автоматически в `data/bot.sqlite3`.
- Превью масок уже лежат в корне проекта: `page_1.jpg` ... `page_5.jpg`.


Важно: в архиве есть скрытый файл `.env`. Если файловый менеджер не показывает скрытые файлы, используй также `env_for_bothost.txt` или `ENV_VISIBLE.txt`, открой их и скопируй содержимое в `.env`.
