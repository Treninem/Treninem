ЗВЕРОБОЙ — сборка для Bothost

Что улучшено в этой версии:
- админ-панель владельца скрыта для всех остальных
- выдача и удаление предметов по названию, reply, @username или ID
- выдача паков по названию, reply, @username или ID
- списки выбора предметов и паков через inline-кнопки
- продажа на рынке и подарки через выбор предмета из своего инвентаря
- вызовы на ПВП работают и по словам "пвп", и по словам "дуэль" / "поединок"
- команды из обычного текста работают через reply и @username
- тексты бонусов и эффектов показаны на русском
- интерфейс старается редактировать прошлое меню и меньше засорять чат

Короткие команды:
/m /p /inv /x /d /s /bm /mk /c /b /task /boss /pet /camp /clan /mail /top /ref /info /tal /fac /ct /adm

Загрузка на Bothost:
1. Загрузи архив
2. Установи зависимости из requirements.txt
3. Убедись, что .env лежит в корне
4. Запусти bot.py

Важно:
- токен уже записан в .env
- так как токен был отправлен в чат, после проверки лучше перевыпустить его у BotFather и заменить в .env


Дополнительные файлы в этой версии:
- item_catalog_ru.txt — полный каталог предметов
- item_catalog_ru.json — тот же каталог в JSON
- game_content_summary.txt — сводка по количеству контента
- command_synonyms_ru.txt — команды и 5 русских синонимов для каждой


Новая версия: в личных сообщениях работает reply-клавиатура навигации, в группах — дуэли, карточки арены и ставки зрителей валютой или материалами.
