# Beast Legends by Treninem

Готовая база проекта под:
- **VK Mini App**
- **единый Python backend**
- **Android-оболочку с синхронизацией по коду**

## Что уже есть
- авторизация через **VK launch params**;
- вход для разработки через `dev_vk_user_id`;
- синхронизируемый профиль;
- выбор персонажа;
- инвентарь, использование предметов, экипировка;
- золото как **основная валюта**;
- магазин, экспедиции, подземелья, крафт, рынок;
- PvP между профилями;
- рейтинг;
- секретная админка;
- создание **новых предметов прямо через админ-панель** с сохранением в SQLite;
- приглашения друзей VK и окно запросов через VK Bridge;
- генерация кода для привязки Android-приложения к тому же аккаунту.

## Куда вставить VK данные
Открой файл:

`backend/config.py`

Вставь данные сюда:

```python
VK_APP_ID = 1234567              # <-- сюда вставь ID VK Mini App
VK_APP_SECRET = 'PASTE_VK_APP_SECRET_HERE'  # <-- сюда вставь защищённый ключ приложения VK
PUBLIC_WEB_URL = 'https://example.com'      # <-- сюда вставь публичный HTTPS URL backend
OWNER_VK_IDS = [123456789]       # <-- сюда вставь свой VK ID
ADMIN_PASSWORD = 'CHANGE_ME_SUPER_SECRET'   # <-- сюда вставь пароль секретной админки
```

## Где менять Android URL
Открой файл:

`android_wrapper/app/src/main/java/com/treninem/beastlegends/MainActivity.kt`

Там есть строка:

```kotlin
backendUrlInput.setText(prefs.getString("backend_url", "https://example.com")) // <-- ВСТАВЬ URL backend по умолчанию
```

## Как запустить backend
Установи зависимости:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python main.py
```

Откроется на:

`http://127.0.0.1:8000`

## Как подключить VK Mini App
1. Разверни backend на публичном **HTTPS** домене.
2. Укажи этот URL в `PUBLIC_WEB_URL`.
3. В настройках VK Mini App укажи этот же URL как адрес приложения.
4. После этого при запуске из VK приложение получит `vk_*` launch params и сможет автоматически логинить игрока.

## Как работает Android синхронизация
1. Игрок заходит в VK Mini App.
2. Нажимает **"Код для Android"**.
3. Получает одноразовый код.
4. В Android-приложении вводит URL backend и этот код.
5. Приложение привязывается к тому же игровому профилю.

## Админка
Только владелец и назначенные админы.

Возможности:
- выдача золота;
- выдача предметов;
- бан / разбан;
- выдача / снятие админки;
- установка уровня;
- создание новых предметов в базе.

## Создание предметов через админку
В панели уже есть готовые выпадающие списки:
- категория;
- редкость;
- тип эффекта;
- изменяемый параметр;
- слот экипировки;
- расходуемость;
- стакаемость.

Тебе остаётся указать:
- новый уникальный ID;
- название;
- описание;
- цифры значений;
- цену;
- вес;
- иконку.
